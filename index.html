Política de Privacidade

Este aplicativo utiliza a API do WhatsApp para fornecer atendimento automatizado aos usuários.

1. Dados Coletados

Coletamos apenas os dados necessários para o funcionamento do atendimento automatizado, como número de telefone e mensagens trocadas durante a conversa.

2. Uso dos Dados

Os dados são utilizados exclusivamente para fornecer respostas automáticas e melhorar a experiência de atendimento via WhatsApp.

3. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros. Toda a comunicação é tratada com confidencialidade.

4. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acessos não autorizados.

5. Contato

Em caso de dúvidas, solicitações ou preocupações com relação à sua privacidade, entre em contato pelo e-mail: contato@seudominio.com.
